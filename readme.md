## git使用的几个步骤和命令：

```shell
git init				#创建git项目
git add .				#添加所有文件到暂存区
git commit -m "描述"	   #提交暂存区的内容到本地仓库 -m 提交信息	
git pull

git status				#查看有改动的文件
```
